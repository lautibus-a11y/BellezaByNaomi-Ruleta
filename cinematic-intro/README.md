# 🎬 Cinematic Intro — Fondo de Pantalla

Coloca aquí el fondo que aparecerá en la pantalla de introducción cinemática.

## Nombre del archivo (IMPORTANTE)
El archivo actual se llama:
```
logo.jpeg
```
→ Ruta completa: `cinematic-intro/logo.jpeg`

También se aceptan estos formatos (cambia la extensión en index.html si usas otro):
- `fondo.webp` ← recomendado (más liviano)
- `fondo.png`
- `fondo.avif`

## Tamaño recomendado
- **Resolución**: 1080 × 1920 px (portrait/vertical)
- **Peso máximo**: 400 KB (usa [Squoosh](https://squoosh.app) para comprimir)
- El efecto Ken Burns hace zoom-in, así que los bordes pueden quedar cortados

## Cómo funciona
1. El intro aparece automáticamente al cargar la web
2. Muestra el fondo con animación lenta de zoom (Ken Burns)
3. El título "BellezzaByNaomi" aparece con animación de lujo
4. Después de ~5 segundos (o al tocar) se hace exit cinemático
5. La app principal aparece debajo

## Sin imagen (fallback)
Si no hay imagen, el fondo usa el gradiente Aurora rosa de la app.
No es necesario añadir una imagen para que funcione.
